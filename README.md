# 📱 App Navigation Compose

## 📌 Descrição do Projeto
Aplicativo desenvolvido em Android utilizando Jetpack Compose com o objetivo de demonstrar a navegação entre telas e a passagem de parâmetros entre elas.

## 🚀 Funcionalidades
- Navegação entre telas
- Uso do NavController
- Passagem de parâmetros entre telas
